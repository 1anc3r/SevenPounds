## 应用简介:
七磅（榜）—— 灵魂的重量。 汇（pa）聚（qu）网络优质信息于七张榜单，为灵魂增重。

## 功能介绍:
1. 见闻: 

    每日: 提供来自知乎社区的精选问答，还有国内一流媒体的专栏特稿
    
    分类: 包括动漫、游戏、财经、电影、音乐、互联网安全等丰富内容
    
    — 数据来源: 知乎日报 
    
2. 图片 : 

    佳人: 每天一张美女图片
    
    美景: 好看的风景照
    
    — 数据来源: Gank.io
    

3. 读书 : 

    书评: 豆瓣读书的**最受欢迎**书评
    
    书榜: 爬取呈现豆瓣图书**TOP250**
    
    — 数据来源: 豆瓣读书 
    
4. 听音 : 

    乐评: 豆瓣音乐的**最受欢迎**乐评
    
    乐榜: 爬取呈现豆瓣音乐**TOP250**
    
    — 数据来源 : 豆瓣音乐 
    
5. 观影 : 

    影评: 豆瓣电影的**最受欢迎**影评
    
    影榜: 爬取呈现豆瓣电影**TOP250**
    
    — 数据来源: 豆瓣电影 
    
6. 视频 : 

    分类: B站各分区**排行榜**
    
    — 数据来源: BiliBili 
    
7. 游戏 : 

    周榜: SteamSpy统计的最近两周玩家数量最多的**TOP100**游戏
    
    分类: Steam各分类**排行榜**
    
    — 数据来源: SteamSpy 

8. 漫画 : 

    推荐: 推荐好看的漫画
    
    排行: 漫画**排行榜**
    
    分类: 以漫画类型分类
    
    — 数据来源: 有妖气漫画
    
9. 编程 : 

    个人: GitHub上Star**最多**的个人
    
    组织: GitHub上Star**最多**的组织
    
    项目: GitHub上Star**最多**的项目
    
    — 数据来源: GithubRanking

## 关于
### 组织
<img src="http://o7gy5l0ax.bkt.clouddn.com/ic_xiyoumobile.png" width = "100" height = "100" alt=""/>

西安邮电大学移动应用开发实验室，亦称西安邮电大学3G手机应用开发实验室，创建于2011年，并被设为“Google 中国大学合作部--Android 人才培养示范基地”。实验室分为苹果iOS、谷歌Android、微软Windows Phone三个小组，是西安邮电大学唯一专注于移动平台应用开发的实验室。

实验室位于西安邮电大学长安校区东区教学楼一层FZ155教室，设有多部移动应用开发设备，可供成员测试相关应用程序。Android开发组还配有由公司提供的Android底层开发箱，可用于开发和测试Android底层程序。
